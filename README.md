# Imba Learn (iOS)

Imba Learn is a Quizlet-inspired iOS study application built with **Swift** and **UIKit**.  
The app allows users to create study modules, manage terms, and practice using flashcards in a clean and simple interface.

The application communicates with a hosted backend API for authentication and data management.

---

## Features

- Sign up and log in
- Create study modules with title, description, and privacy settings
- Add, edit, star, and delete terms inside a module
- Study terms using flashcards mode
- View and manage personal modules from the dashboard
- Update account password or delete account

---

## Project Goal

The goal of this project is to provide a **modern and focused iOS alternative to Quizlet**, emphasizing:
- Fast and intuitive navigation
- Clear user interface
- Effective repetition and recall through study modes

---

## Tech Stack

- **Language:** Swift  
- **UI Framework:** UIKit  
- **Architecture:** MVVM  
- **Networking:** URLSession  
- **Dependency Management:** Swift Package Manager (SPM)

---

## How to Run the Project

### Prerequisites
- macOS with Xcode 15.0 or later
- iOS 16.0 or later simulator/device
- Swift 5.9 or later

### Installation Steps

1. **Clone the repository**
   
   ```bash
   git clone https://github.com/yourusername/imba-learn-ios.git
   cd imba-learn-ios
2. Open the project in Xcode
    * Open ImbaLearn.xcodeproj in Xcode
    * Or use the command line: open ImbaLearn.xcodeproj
3. Select a simulator or device
    * Choose a simulator from the device selector (top bar)
    * Recommended: iPhone 15 Pro (iOS 17.2+)
4. Build and run the app
    * Press Cmd + R to build and run
    * Or click the play button ▶️ in Xcode toolbar
